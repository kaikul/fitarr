Privacy Policy for Fitarr

Effective Date: 2025-06-11

Fitarr is an open-source fitness tracking application created to empower individuals to manage their own health and wellness data independently. This project is maintained by contributors and is distributed freely under an open-source license.

1. No Data Collection

Fitarr does not collect, store, or transmit any personal data to our servers. All data entered into the app remains on the user’s device or server.

2. User-Controlled Integrations

If you choose to connect third-party services (e.g., Garmin, Apple Health, Fitbit), all data fetched via these integrations is handled locally and privately on your own system. We do not access or share any integration data.

3. No Tracking or Analytics

This application contains no embedded analytics, tracking pixels, or telemetry. We do not monitor usage, user behavior, or system performance.

4. Your Responsibility

Because Fitarr is self-hosted, the responsibility for data security, storage, and backups lies with the user or the entity hosting the application.

5. Open Source Nature

This application is provided as-is, without warranty. Developers and maintainers take no responsibility for how the software is used or hosted. Review the code and configure integrations responsibly.
